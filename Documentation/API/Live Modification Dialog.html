LSS.lvlib:Live Modification Dialog.vi

This dialog box can be used to help generate style sheets and observe their effect on the user interface. The editor will populate with the default values for each LSS style when an item from "Available Properties" is double clicked.

Front Panel Objects

Although the editor is setup to allow a user to select an active VI to edit, this input can be used to pass in Front Panel Objects if the references are already open when usinng this VI as a pop up dialog.

    Generic front panel object reference

Error in

error in can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

Error out

Returns errors from this VI if any occured.
